ACCC Site Skeleton

This is a minimal React + Vite + Tailwind skeleton for the ACCC Cloud Club site.

Quick start:

1. Install dependencies:

```bash
cd accc-site
npm install
```

2. Run dev server:

```bash
npm run dev
```

Replace the timeline placeholder in `src/pages/Events.jsx` with your provided timeline image.

Brand colors (from provided logo SVG): primary `#2074D5`, dark `#003181`.
